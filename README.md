# My Cypress Project!

This project utilize cypress to automate the test of a login feature and a user update feature

## Instalation!

```bash
npm install
```
>**NOTE:**
>
>It's necessary to have **node** installed

## Running:


## You open the Cypress UI
```bash
npx cypress open
```

## Run the cypress by comand line:
```bash
npx cypress run
```
